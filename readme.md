## 📚 What I'm Learning

### 🟦 C Fundamentals

- Basic syntax and program structure
- Data types and variables
- Constants and macros
- Input and output with `printf` and `scanf`
- Operators and expressions
- Control flow: `if`, `switch`, `for`, `while`, `do-while`
- Functions and modular code

### 🧠 Memory and Data Structures

- Pointers and pointer arithmetic
- Arrays and strings
- Structs and unions
- Dynamic memory allocation with `malloc`, `calloc`, `realloc`, and `free`
- Passing data by value vs reference

### 🛠️ Advanced Topics

- File handling with `fopen`, `fread`, `fwrite`, and `fclose`
- Error handling and return codes
- Build process and compilation flags
- Working with headers and separate source files
- Debugging and tracing runtime behavior
